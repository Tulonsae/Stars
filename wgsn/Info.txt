Information for stars.wgsn
==========================

Description
-----------
This database table uses the IAU-CSN.txt data file from the IAU WGSN. In 2016,
the International Astronomical Union (IAU) organized the Working Group on Star
Names (WGSN), consisting of an international group of astronomers, to catalog
and standardize proper names for stars for the international astronomical
community. Please see their Terms of Reference webpage (see Other Links below,
under Location) for more info. As of June 30, 2016, the list contains 58 stars.

Files
-----
Info.txt		this file
schema.sql		schema file for table, see Schema below
create-input-sql.awk	awk script - converts data into input sql

External Files
--------------

Instructions
------------
From Source:
  To get the IAU-CSN.txt file
     curl -O http://www.pas.rochester.edu/~emamajek/WGSN/IAU-CSN.txt
  To create the input.sql
     tail -n +5 IAU-CSN.txt | awk -f create-input-sql.awk > input.sql
From Tulonsae:
  To get the External Files
     curl -O https://tulonsae.org/wgsn/30-Jun-16_IAU-CSN.txt \
          -O https://tulonsae.org/wgsn/input.sql

Data:
-----
  Name - official name
  Designation - cat code + its cat ID, usually HR (Harvard Revised)
  RA(J2000) - Right Ascension, J2000, in degrees (converted to decimal)
  Dec(J2000) - Declination, J2000, in degrees (converted to decimal)
  Approved - date (mm/d/yy)
  Vmag - apparent visual magnitude
  ID - ??, 3-letter Greek and/or 2-digit number, if known
  Con - constellation
  HIP# - Hipparcos Catalog ID
  HD# - Henry Draper Catalog ID

Schema:
-------
Table name: wgsn
  Name		varchar(16)*	from Name field
  Desg		varchar(16)*	from Designation field
  RA		float*		from RA(J2000) field
  Decl		float*		from Dec(J2000) field
  Approved	date*		from Approved field, converted to yyyy-mm-dd
  Vmag		float		from Vmag field
  ID		varchar(8)	from ID field
  Con		varchar(8)	from Con field
  HIP		unsigned int	from HIP# field
  HD		unsigned int	from HD# field
  HR		unsigned int	from Designation field, only if cat code HR
Note: * Indicates NOT NULL, otherwise DEFAULT NULL.
